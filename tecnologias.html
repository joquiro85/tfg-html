Tecnologías Utilizadas

Terraform

Herramienta para infraestructura como código (IaC) usada para crear y gestionar los recursos en la nube.

Amazon Web Services (AWS)

Plataforma de servicios en la nube donde se desplegaron todos los componentes del proyecto.

GitHub

Repositorio remoto utilizado para almacenar el código fuente y controlar versiones del proyecto.

Visual Studio Code

Editor de código principal utilizado para el desarrollo de la infraestructura con Terraform.

Nginx

Servidor web ligero usado como proxy inverso para gestionar el tráfico hacia las instancias.
